PizzaWars
=========

A 2-player platformer fighting game built with Greenfoot.

Gameplay
--------
Two players battle by throwing pizza slices at each other.
Each player has 3 lives. Last one standing wins!

Controls
--------
Red Player (left side):  W - Jump, A - Move Left, D - Move Right, S - Shoot Pizza
Green Player (right side): UP - Jump, LEFT - Move Left, RIGHT - Move Right, DOWN - Shoot Pizza
