# PizzaWars

A 2-player platformer fighting game built with Greenfoot.

## Gameplay

Two players battle by throwing pizza slices at each other. Each player has 3 lives. Last one standing wins!

## Controls

| Action | Red Player | Green Player |
|--------|-----------|-------------|
| Jump | W | UP |
| Move Left | A | LEFT |
| Move Right | D | RIGHT |
| Shoot Pizza | S | DOWN |
